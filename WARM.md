## Призначення та персона

```yaml

tags: [purpose]
status: active

Бот для управління домашнім господарством Саші і Ксю. Основні функції: ведення списку покупок, рецептів, нагадувань; інтеграція з супермаркетом Metro; голосові повідомлення; фото-рецепти; URL-рецепти. Ім'я — Меггі. Персональний, не для клієнтів.

last_touched: 2026-04-23
tags: [architecture, tool-use]

```

Повна міграція з JSON-парсингу тексту на нативний tool-use Anthropic API. 9 інструментів замінюють усю раніше-JSON логіку: додавання/видалення зі списку покупок, збереження рецептів, управління нагадуваннями, інтеграція з Metro. ## Metro supermarket integration

```yaml
last_touched: 2026-04-23
tags: [metro, integration]
status: active
```

`core/metro.py` — окремий модуль для роботи з мережею Metro. Функції: пошук товарів, автозаповнення кошика за списком покупок, аналіз патернів минулих покупок, `suggest_missing_items()` — рекомендує товари на основі історії. Додає "магазинну" розумність до списку покупок.

## Voice + photo handling

```yaml
last_touched: 2026-04-23
tags: [input, voice, photo]
status: active
```

Голосові повідомлення обробляються через faster-whisper (локально на Pi5). Рецепти можна додавати трьома способами: (1) текст, (2) фото (розпізнавання інгредієнтів + кроків через Vision), (3) URL (парсинг сторінки рецепту).
